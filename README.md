# 🧮 Calculator Website

A simple and responsive Calculator Website built using HTML, CSS, and JavaScript. This project performs basic arithmetic operations and demonstrates fundamental web development concepts.

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Decimal support
- Clear (C) button
- Delete (⌫) button
- Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Calculator/
│── index.html
│── style.css
│── script.js
└── README.md
```

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it with Live Server in VS Code.
